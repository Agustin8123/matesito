Sobre Matesito...

Matesito surgió como una idea loca, un día simplemente pensé: "¿Será muy dificil hacer una red social?" y acto seguido empecé a hacer una para responderme mi pregunta.

Matesito utiliza los siguientes servicios de terceros:

Render:

Render es una empresa que, entre otras cosas, brinda un muy buen alojamiento de servicios web. Es decir, Matesito funciona en sus servidores. También es donde está alojada actualmente la base de datos que guarda todos los textos que se suben en matesito. Actualmente está funcionando con el plan gratuito, y esta es la razón por la que en la URL de Matesito dice: "onrender".

Cloudinary:

Es una empresa que brinda un servicio para almacenar archivos, es donde se guardan todas las fotos, videos y audios que se suben en Matesito. Actualmente está funcionando con el plan gratuito.

MicroReact:

Es el sistema de reacciones de Matesito, es de de código abierto y fue desarrollado por Thiago Rangel.
Matesito utiliza una versión modificada de MicroReact. Esta modificación consiste en simplemente funcionar con un contador interno de la API de Matesito, en lugar de el contador de Jason Cameron, Abacus, con el que funcionaba originalmente.
