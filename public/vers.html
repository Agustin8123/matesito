Versión 1.0.8 (actual)

Se mejoró el menu desplegable que se agregó la versión anterior. y ahora el logo funciona como el el boton de inicio.

Versión 1.0.7

Ahora el botón de usuario funciona como un menu desplegable, que contiene todas las opciones que anteriormente estaban en el header de la pagina. Esto se ha hecho para solucionar la mayoria de bugs en dispositivos moviles.

Versión 1.0.6

Se implementó una lista de palabras baneadas que sera actualizada regularmente. Esto para evitar spam y contenido inapropiado en la plataforma.

Versión 1.0.5

Para la comodidad de los usuarios, se agregó un casilla de recordar sesión en la pestaña de sesión. También agregó una casilla de aceptación de los Términos y Condiciones y Política de Privacidad en la pestaña de creación de cuentas.
Se inhabilitaron los botones de selección de archivos para prevenir bugs relacionados con la base de datos.

Versión 1.0.4

Se implementó un sistema que evita que los usuarios repitan una y otra vez el mismo post.

Versión 1.0.3

Añadió los Términos y Condiciones asi como también la Política de Privacidad.
Se cambió la pagina de inicio de "Iniciogv" a "inicio". Ahora "iniciogv" se usa como un centro de redireccionado.

Versión 1.0.2

Solucionó algunos bugs visuales, agregó el listado de versiones, y la función de redireccionar a la pagina de mantenimiento si así se requiera.

Versión 1.0.1

Igual que la versión 1.0, pero con el error de seguridad corregido.

Versión 1.0

La primera versión publica, con sistema de almacenamiento de datos y que permitía la interaccion online.
Tenía un error grave de seguridad. Este permitía crear varias cuentas con nombres iguales. Fue rapidamente corregido, dando lugar a la versión 1.0.1

Versión Preliminar

Esta versión fue el primer plasmado de la idea del proyecto. Solo funcionaba localmente en el navegador, y no tenia ningun tipo de interaccion con usuarios de otros dispositivos.
